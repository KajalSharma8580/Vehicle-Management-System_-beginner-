# 🚗 Vehicle Management System (C++)

A beginner-friendly Object-Oriented Programming project developed in C++ to demonstrate the concept of **Inheritance**.

## Features

- Base class: Vehicle
- Derived classes:
  - Car
  - Bike
  - Truck
- Start Vehicle
- Stop Vehicle
- Display Vehicle Details

## OOP Concepts Used

- Classes & Objects
- Constructors
- Inheritance
- Protected Access Specifier
- Member Functions

## Technologies

- C++
- VS Code
- MinGW Compiler
